# Ingeniería de Software I 

Repositorio con material práctico de la materia **Ingeniería de Software I** de 2° año.

---

## 📚 Objetivos

Este repositorio contiene ejercicios, plantillas y diagramas relacionados con los siguientes temas de la materia:

- Proceso de software y ciclos de vida de software.
- Ingeniería de requerimientos y métodos formales.
- Metodologías de análisis y diseño, lenguajes de modelado.
- Sistemas de tiempo real.
- Conceptos de calidad de software.
- Teoría General de Sistemas y Sistemas de Información.
- Privacidad, integridad y seguridad en sistemas de información.

Cada ejercicio incluye documentación, diagramas UML y plantillas de práctica según corresponda.

---

## 🛠 Herramientas usadas

- **Google Sheets** – Para plantillas de historias de usuario y casos de uso.  
- **Draw.io** – Para diagramas UML, flujos de datos y de control.  
- **PDF / Markdown** – Para guardar y mostrar los ejercicios y consignas.
